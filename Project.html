How it works
  • Everything is saved automatically to this browser's local storage — no account, no server, no internet needed.
  • Click a topic's checkbox to cycle it: Not started → In progress → Completed. Progress rolls up to units, subjects and the overall gauge.
  • When a topic is finished, hit ✓ Commit to seal it. Sealed topics get a gold badge and can't be flipped back by a stray tap — use the ↺ unlock if you need to reopen one.
  • Completing a topic is logged so the dashboard can estimate your pace and forecast a finish date.
  • To move this app or its data, copy this HTML file (it carries the code) and export a JSON file (it carries your progress).
